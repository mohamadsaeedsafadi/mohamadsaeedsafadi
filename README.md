# Portfolio — Mohammed Saeed Safadi (Improved)
ملف يحتوي على نسخة محلية من بورتفوليو مُحسّن وجاهز للرفع إلى GitHub Pages.

### هيكل الملفات
- index.html — الصفحة الرئيسية
- style.css — أنماط CSS
- script.js — جافاسكربت لتحسين التفاعلية
- files/CV-Mohamad-Safadi.pdf — (مُؤشر مكان يمكنك استبداله بسيرتك الحقيقية)
- README.md — هذا الملف

### تعليمات رفع على GitHub Pages
1. أنشئ مستودعًا جديدًا (مثلاً `mohamadsaeedsafadi.github.io`).
2. ارفع محتويات هذا المجلد إلى الفرع `main`.
3. إن لم تعمل الصور أو ملفات PDF، تأكد أن مسار `/files/` موجود وأن الملفات مرفوعة.

### تخصيص سريع
- افتح `index.html` وعدِّل النصوص (الاسم، مواقع التواصل، وصف المشاريع).
- أضِف صورًا حقيقية واستبدل ملف السيرة الذاتية في `/files/CV-Mohamad-Safadi.pdf`.
- لإضافة مشاريع جديدة، انسخ إحدى بطاقات `.project-card` وعدل المحتوى.

إذا تريد، أستطيع الآن:
- دمج محتويات مشروع `complaint-system` (قراءة README ولقطات) وعرض مقتطفات (إن رغبت).
- تحويل التصميم إلى React أو إضافة CMS لسهولة التحديث.
